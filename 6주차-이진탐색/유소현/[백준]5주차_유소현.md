## 1427번 소트인사이드
```python
N = int(input())
N_list = []

for i in N:
    N_list.append(i)

N_list.sort(reverse=True)
print("".join(map(str, N_list)))
```


## 2012번 등수매기기
```python
import sys
N = int(sys.stdin.readline())

N_list = []
for i in range(N):
    N_list.append(int(sys.stdin.readline()))

N_list.sort()

d = 0
for j in range(1, N+1):
    d += abs((j - N_list[j-1]))

print(d)
```
